## 模組一：Prompt 心法 — RICE 框架

### 為什麼你用 AI 寫出來的東西「很普通」？

因為你給的指令太模糊了。

| 你說的 | AI 聽到的 | 結果 |
|--------|----------|------|
| 「幫我寫一段訊息」 | 寫給誰？什麼目的？ | 一段四不像的通用內容 |
| 「幫我寫一段 LINE 開發訊息給做餐飲的客戶，推薦我們的食材配送服務，強調隔日到貨和免運門檻低」 | 清楚知道對象、產品、賣點 | 一段能直接傳出去的訊息 |

### RICE 框架 — 四個要素讓 AI 變成你的業務助理

#### R — Role（角色）

告訴 AI 它是誰，它會用那個角色的語氣和專業來回答你。

| 情境 | 角色設定 |
|------|---------|
| 寫開發訊息 | 「你是一位有 10 年經驗的業務經理」 |
| 寫社群文案 | 「你是一位熟悉社群行銷的文案專家」 |
| 整理會議記錄 | 「你是一位專業的業務助理」 |
| 回覆客戶疑慮 | 「你是一位擅長客戶關係管理的客服主管」 |

#### I — Instruction（指令）

明確說出你要 AI 做什麼。一次一件事，不要貪心。

```
❌ 幫我寫訊息還有做一張圖
✅ 請幫我撰寫一段 LINE 客戶開發訊息
```

#### C — Context（背景）

給 AI 足夠的資訊，讓它寫出來的東西「像你寫的」而不是「像 AI 寫的」。

背景資訊越多，產出越精準：

- 你的公司 / 品牌名稱
- 你賣什麼（產品或服務）
- 目標客戶是誰（產業、職位、規模）
- 客戶的痛點是什麼
- 你的獨特賣點（為什麼選你不選別人）
- 這段訊息 / 這張圖的用途

#### E — Example（範例）

給 AI 一個參考，它就會模仿那個風格。

```
語氣參考：「親切但專業，不要太推銷，像朋友推薦好東西的感覺」
格式參考：「第一段打招呼，第二段講痛點，第三段講解決方案，最後一段 call to action」
長度參考：「控制在 150 字以內，適合 LINE 閱讀」
```

### RICE 完整範例

```
【R】你是一位有 10 年經驗的業務經理，擅長用溫暖但專業的語氣開發新客戶。

【I】請幫我撰寫一段 LINE 客戶開發訊息。

【C】
- 我的公司：美味食材有限公司
- 我們的服務：餐廳食材配送，主打有機蔬菜和進口肉品
- 目標客戶：中型餐廳老闆（月營業額 50-200 萬）
- 客戶痛點：現在的供應商常常缺貨、品質不穩定
- 我們的優勢：隔日到貨、品質不滿意免費換貨、滿 3000 免運

【E】
- 語氣：親切專業，像朋友推薦好東西
- 格式：開頭問候 → 點出痛點 → 介紹方案 → 行動呼籲
- 長度：150 字以內（適合 LINE 閱讀）
```

### 進階技巧

| 技巧 | 怎麼用 | 範例 |
|------|--------|------|
| 迭代修改 | 叫 AI 調整不滿意的地方 | 「語氣再親切一點，少一點推銷感」 |
| 格式指定 | 指定輸出的格式 | 「用表格整理」「分成三個版本讓我選」 |
| 批量處理 | 一次處理多筆 | 「以下五位客戶的資訊，分別寫客製化開發訊息」 |
| 翻譯改寫 | 你寫中文，AI 翻成英文 | 「把這段訊息翻成英文，語氣保持一樣」 |

## 文字生成：LINE 開發訊息 & 客戶回覆

### 情境一：LINE 客戶開發訊息

#### 步驟

1. 打開 ChatGPT（chat.openai.com）或 Gemini（gemini.google.com）
2. 把 RICE 框架填好，貼進去
3. 看產出結果，不滿意就迭代修改
4. 滿意後複製，貼到 LINE 傳給客戶

#### Prompt 模板：LINE 客戶開發訊息

```
你是一位有 10 年經驗的業務經理，擅長用溫暖但專業的語氣開發新客戶。

請幫我撰寫一段 LINE 客戶開發訊息。

背景資訊：
- 我的公司：[你的公司名]
- 我們的產品/服務：[你賣什麼]
- 目標客戶：[客戶是誰，什麼產業]
- 客戶常見痛點：[他們遇到什麼問題]
- 我們的優勢：[為什麼選你]

要求：
- 語氣親切專業，不要有推銷感
- 格式：問候 → 點出痛點 → 介紹方案 → 行動呼籲
- 長度 150 字以內（適合 LINE 閱讀）
```

### 情境二：客戶回覆（處理疑慮）

當客戶 LINE 回覆說「有興趣但是...」，你要專業地打消他的疑慮。

#### Prompt 模板：回覆客戶疑慮

```
你是一位資深的客戶關係經理，擅長用同理心回覆客戶的疑慮，讓客戶感到被理解和安心。

以下是客戶的 LINE 訊息，請幫我寫一段專業回覆：

客戶訊息：
「你好，你們的產品看起來不錯，但我目前跟現在的供應商合作很久了，
不太確定換供應商值不值得，而且你們的價格好像比較高？」

回覆要求：
- 先同理客戶的顧慮（不要否定他）
- 用具體數據或案例說明我們的價值
- 提出低風險的嘗試方案（例如：免費試用、小量訂購）
- 語氣溫暖、不施壓
- 150 字以內（適合 LINE 閱讀）

我們的優勢資訊：[在這裡補充你的賣點]
```

## 模組二：圖片生成 — 不會設計也能做宣傳圖

### 你需要的不是設計師，是一句好的圖片描述

#### 圖片 Prompt 公式

```
[主體/內容] + [風格] + [色調] + [用途/尺寸] + [額外要求]
```

#### 範例

| 用途 | Prompt |
|------|--------|
| FB 貼文 | 「一張食材配送服務的宣傳圖，新鮮蔬菜和肉品排列在木桌上，溫暖自然的色調，適合 Facebook 貼文的正方形比例，上方留白空間可以放標題文字」 |
| 活動海報 | 「一張商會聯誼餐會的宣傳海報，優雅的餐桌佈置，金色和深藍色調，正式但溫馨的氛圍，直式 A4 比例」 |
| LINE 推播 | 「一張年終促銷的宣傳圖，喜氣的紅金配色，簡潔有力，橫式比例，主體是禮盒和折扣標籤」 |

### 操作步驟（Gemini + Nano Banana Pro）

> Gemini 最新的圖片生成模型叫 **Nano Banana Pro**，中文字顯示精準、支援多種比例，完全免費。

1. 打開 Gemini（gemini.google.com）
2. **重要：選擇「2.5 Pro」思考型模型**（看到香蕉圖示就對了）
3. 直接輸入圖片描述（中文就可以）
4. Gemini 會產出圖片，不滿意可以要求修改
5. 滿意後點擊下載

#### 指定圖片比例（很實用！）

在 Prompt 最後加上比例要求：

| 用途 | 比例 | 怎麼說 |
|------|------|--------|
| Facebook / IG 貼文 | 1:1 | 「正方形比例」 |
| LINE 推播 / 簡報配圖 | 16:9 | 「橫式 16:9」 |
| 手機直式海報 | 9:16 | 「直式 9:16」 |
| IG 人像照片 | 4:5 | 「比例 4:5」 |
| A4 海報 | 3:4 | 「直式 3:4」 |

> 解析度可以要求「4K 畫質」讓圖片更清晰。

## 模組三：簡報生成 — 10 分鐘做完提案簡報

### Gamma — 輸入文字就給你簡報

Gamma（gamma.app）可以把一段文字直接變成漂亮的簡報，不需要排版、不需要找模板。

### 操作步驟

1. 打開 gamma.app（免費註冊）
2. 點選「Create new」→「Paste in text」
3. 貼上你的產品/服務介紹（3-5 句話就夠）
4. 選擇簡報風格（商務、簡約、活潑...）
5. Gamma 自動生成完整簡報（含圖片、排版）
6. 微調：點任一頁就能編輯文字、換圖、改版面
7. 匯出：下載 PDF 或 PPTX

### 你要準備什麼？

只要一段話就好，包含：

```
[你的公司名] 提供 [什麼產品/服務]，
專門服務 [目標客戶]。
客戶常遇到 [什麼痛點]，
我們的解決方案是 [你的方案]，
優勢在於 [1. xxx 2. xxx 3. xxx]。
```

#### 範例

```
美味食材有限公司提供餐廳食材配送服務，
專門服務中型餐廳（月營業額 50-200 萬）。
餐廳老闆常遇到供應商缺貨、品質不穩定的問題，
我們提供隔日到貨的穩定配送，
優勢在於：1. 品質不滿意免費換貨 2. 滿 3000 免運 3. 專屬業務一對一服務。
```

### 進階用法：用 AI 先幫你寫簡報大綱

如果你不確定簡報要放什麼內容，可以先問 ChatGPT / Gemini：

```
我要做一份客戶提案簡報，用來向 [目標客戶] 介紹我們的 [產品/服務]。
請幫我列出簡報大綱，每頁的標題和重點，大約 6-8 頁。
```

拿到大綱後，再丟進 Gamma 生成簡報。

## 模組四：影片生成 — Seedance 2.0，AI 影片一句話就能生

### Seedance 2.0 是什麼？

字節跳動旗下**即夢平台**推出的 AI 影片生成模型。輸入文字描述或上傳圖片，就能產出高品質影片，不需要攝影師、不需要剪接師。

| 能力 | 說明 |
|------|------|
| 文字生影片 | 打一段描述就產出影片 |
| 圖片生影片 | 上傳產品圖讓它動起來 |
| 角色一致性 | 同一人物跨鏡頭不會變臉 |
| 精準對嘴 | 上傳語音自動對嘴型 |
| 物理模擬 | 碰撞、光影自然不穿幫 |

**輸出規格：** 原生 1080p（付費可 2K）、支援 16:9 / 9:16 / 1:1、長度 5-12 秒

### 五要素 Prompt 公式

跟圖片一樣，影片 Prompt 越具體，產出越好：

| 要素 | 說明 | 範例 |
|------|------|------|
| **場景** | 環境與時間 | 「雨天的城市街道，霓虹燈閃爍」 |
| **主體** | 人物或產品特徵 | 「一位穿西裝的男子手持咖啡杯」 |
| **動作** | 主體的具體行為 | 「拿起產品喝了一口，露出滿意微笑」 |
| **鏡頭** | 攝影機運動 | 「從遠景推到近景特寫、低角度拍攝」 |
| **氛圍** | 光影與色調 | 「電影質感、暖色調、柔和背景虛化」 |

### 操作步驟

1. 打開 即夢平台（jimeng.jianying.com）
2. 註冊登入（免費）
3. 選擇「AI 影片」功能
4. 上傳產品圖片（可多張）
5. 用五要素公式寫 Prompt
6. 選比例（9:16 手機直式 / 16:9 橫式）
7. 點擊生成 → 等待 → 下載

### @ 引用技巧

上傳圖片後，用 **@圖片** 引用素材，AI 就會照著你的圖來生成：

```
@圖片1 的產品放在木桌上，
鏡頭從遠景慢慢推進到特寫，
暖色調光線灑落，
電影質感，背景柔和虛化
```

> 多張圖可以分別 @圖片1 @圖片2 指定不同畫面

### 完整 Prompt 範例

```
場景：明亮的現代辦公室，陽光透過大片落地窗灑入
主體：@圖片1 的產品放在白色桌面上，旁邊有一杯咖啡和筆記本
動作：一隻手從畫面右側伸入拿起產品，鏡頭跟隨手部動作
鏡頭：從俯拍慢慢轉為側面平視，最後推到產品特寫
氛圍：自然光、暖色調、清新專業感、淺景深
```

## 模組五：會議記錄 — 開完會 5 分鐘搞定紀錄 + LINE 跟進訊息

### 痛點

開完客戶會議，你通常：
- 憑記憶整理 → 漏東漏西
- 拖到隔天才寫 → 細節忘了
- 客戶等你的 follow-up → 你還在整理

### 新做法：錄音 → 轉文字 → AI 整理 → LINE 跟進訊息自動產出

```
開會時按錄音  →  手機自動轉文字  →  貼進 AI 整理  →  5 分鐘內用 LINE 傳出跟進訊息
```

### 步驟 1：錄音 + 轉文字

| 工具 | 說明 |
|------|------|
| **iPhone 語音備忘錄** | iOS 18+ 支援即時轉錄，錄音完直接有文字稿 |
| **Google Recorder** | Android 使用者推薦，免費，自動轉文字 |
| **其他錄音 App** | 任何錄音 App 都可以，之後再用 AI 轉文字 |

> 提醒：請事先告知與會者會錄音，尊重隱私。

### 步驟 2：用 AI 整理

把轉出來的文字稿，貼進 ChatGPT 或 Gemini，搭配以下模板：

#### Prompt 模板：客戶會議紀錄

```
你是一位專業的業務助理，擅長整理會議內容並規劃後續行動。

以下是我跟客戶開會的逐字稿，請整理成以下格式：

1. 會議基本資訊
   - 日期 / 時間
   - 客戶名稱 / 公司
   - 與會人員

2. 客戶需求摘要（條列式）

3. 客戶的疑慮或待回覆問題

4. 我方承諾事項

5. 後續待辦
   | 項目 | 負責人 | 期限 | 優先順序 |
   |------|--------|------|---------|

6. 建議的 LINE 跟進訊息草稿（150 字以內，語氣專業溫暖）

逐字稿如下：
[貼上逐字稿]
```

### 步驟 3：確認 + 傳出

1. 檢查 AI 整理的內容是否正確（AI 有時會理解錯誤）
2. 調整跟進訊息的內容
3. 直接用 LINE 傳出 — 開完會 5 分鐘內 follow-up
